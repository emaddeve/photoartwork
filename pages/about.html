photoartwork is a web application created by Emad Al Rifai as a M2-DNR2Iat UNICAEN with a gool of making a simple image gallery without database.


Application Map

HOME PAGE: present some photo with slideshow way with some details about each photo.

DETAILS PAGE: present the image who has been selected with it's (EXIF,XMP,IPTC) METADATA with the ability to download the image itself and it's XMP sidecar alongside the option to search the image on Flickr.

UPLOAD PAGE:this page allow the user to upload his own image

UPLOADEDIMAGE PAGE: this page allow the user to modify the metadata of the image which he uploaded.

Technology Description

this project was developed using PHP for applying exiftool commands on the images and javascript for some functionality

Technology By Page

HOME PAGE: the home page load some images from images folder and execut an exiftool command to get the Creator and Title and show on the corresponding image with a link for each image point to the details page for this image

DETAILS PAGE: it contain the metadata of the image which was extracted by PHP command (exec shell) with exiftool the metadata are divided to 3 category EXIF,XMP,IPTC and each category have a show up button by click on it we can see the corresponding data in additional to the download buttons one for the image itself and other for the XMP sidecar of the image which was made by the propriety (DOWNLOAD)of the element (a) and a search option which search an image on Flickr by the chosen criteria suggested by a dropdown list these criteria was taken from the XMP data of the image itself by the command XMP of exiftool

UPLOAD PAGE:it consist of html form who permit of choose an image by browsing . and send the information to the photoupload.php to save the image in the image folder and send it's information again to uploadedImg.php where we can see the metadata in a formula permit of changing the data by sending it's information to edit.php
